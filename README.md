# Descrição do Projeto

O projeto realizado é uma representação do jogo Connect4 utilizando um arduino. O projeto foi feito utilizando duas matrizes de LED simulando o "tabuleiro" de cada jogador, o jogo acontece da seguinte forma, cada jogador joga por turnos sua peça e seu objetivo é fazer com que se alinhem (para qualquer direção) quatro de suas peças, a dificuldade é que o oponente pode colocar uma peça dele visando atrapalhar a estratégia rival. Para a nossa versão, acrescentamos uma nova dificuldade: um jogador não consegue ver as peças do outro, apesar de elas continuarem interagindo entre si, o que traz um novo nível estratégico para a partida.
